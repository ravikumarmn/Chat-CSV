# Chat With Your CSV 


**Requirements**
- pip
- python3
- llamaindex


It is recommended to create and activate virtual environment:
```
$ python -m venv .venv

# LINUX
$ . .venv/bin/activate

# WINDOWS
$ . .venv/Scripts/activate
```

**Install python requirements**
```
$ pip install -r requirements.txt
```

**Usage**

Run the below command in the terminal.

```
$ streamlit run app.py
```
